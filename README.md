# Scoutly

Youth sports recruiting and social platform, starting with volleyball.

## Demo Athlete
Jake Wolf — Libero #3, Class of 2029, Rockstar Volleyball 16U
- National Rank: #89 | CA Rank: #11
- Stats: 94% dig pct, 4.2 digs/set, 98% pass pct, GPA 3.8

## Stack
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Vercel deployment

## Features
- **Profile Tab** — Athlete card, stats, rankings, highlight reel, coach endorsements
- **Recruiting Tab** — School pipeline tracker, email generator, follow-up tips
- **Feed Tab** — Social posts, news cards, post composer with broadcast toggles

## Getting Started

npm install
npm run dev

Open http://localhost:3000

## Deploy
Push to GitHub, import to Vercel. No env vars needed.
